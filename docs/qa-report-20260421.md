### 🟡 Minor

- **[q-01]** `/api/galleries` `activeCount` is computed with `new Date()` and cached `s-maxage=300` — a gallery whose only exhibition starts or ends inside that 5-minute window may return a value one day stale. Acceptable for an informational badge; documented here so it doesn't surface as a regression later. Mitigation idea: promote to `revalidate = 60` if field becomes authoritative.
  Location: `app/api/galleries/route.ts:14-19`.

- **[q-02]** The `/api/exhibitions` and `/api/galleries` routes set `Cache-Control: public, ... s-maxage=300` so every unique `?q=...` produces its own CDN cache entry. A hostile loop over random queries could bloat the CDN cache but cannot pierce origin (Zod caps `q` at 80 chars and `limit` at 100). No rate-limit layer is planned for MVP; re-evaluate when we move off static JSON.
