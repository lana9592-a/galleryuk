# GalleryUK — User Stories (v1.0)

> 포맷: **As a <role>, I want <goal>, so that <benefit>.** + Acceptance Criteria
> ID 체계: `US-<번호>` / Priority: Must(M) / Should(S) / Could(C)

---

## 탐색 & 목록

### US-01 현재 진행 중 전시 탐색 [M]
**As a** 런던 여행자,
**I want to** 현재 진행 중인 전시를 한 목록에서 훑어볼 수 있다면,
**so that** 이번 주 방문할 전시를 빠르게 결정할 수 있다.

- [ ] 홈 진입 시 "Now on" 탭이 기본 선택
- [ ] 카드에 **제목 / 미술관명 / 기간(~DD MMM) / 썸네일**이 표시
- [ ] 카드 탭 → 전시 상세로 이동
- [ ] 스크롤로 30개 단위 추가 로드 또는 무한 스크롤
- [ ] 오프라인 시 최근 캐시된 목록이 노출되며 배너로 "오프라인 보기" 고지

### US-02 곧 시작하는 전시 보기 [M]
**As a** 현지 애호가,
**I want to** "Coming Soon" 탭에서 앞으로 2개월 내 개막 전시를 볼 수 있다면,
**so that** 일정을 미리 잡을 수 있다.

- [ ] "Coming Soon" 탭에서 startDate가 오늘 ~ 오늘+60일인 항목만
- [ ] "Starts in N days" 라벨 표시
- [ ] 최신 개막 순 기본 정렬

### US-03 카테고리 필터 [S]
**As a** 사진 전공 학생,
**I want to** "Photography" 카테고리만 필터링할 수 있다면,
**so that** 관심 없는 전시를 건너뛸 수 있다.

- [ ] 필터 칩(카테고리) 탭 → 즉시 목록 갱신
- [ ] 다중 선택 가능 (OR)
- [ ] URL 쿼리에 반영되어 공유 가능 (`?category=photo,sculpture`)
- [ ] 모바일에서 스크롤 시 필터 칩이 sticky

### US-04 기간/지역 필터 [S]
**As a** 주말 한정 방문자,
**I want to** "이번 주말 + South London"만 필터링하고 싶다,
**so that** 이동 시간을 줄일 수 있다.

- [ ] 기간: 오늘 / 이번 주 / 이번 주말 / 이번 달 / 직접 선택
- [ ] 지역: 런던 내 자치구(Borough) 또는 도시(영국 전역 확장 시)
- [ ] 필터 조합 시 URL 쿼리 누적

## 상세 & 예매

### US-05 전시 상세 정보 확인 [M]
**As a** 방문 결정 직전 사용자,
**I want to** 전시 상세에서 기간·장소·가격·설명을 한 화면에서 볼 수 있다면,
**so that** 추가 검색 없이 갈지 말지 결정할 수 있다.

- [ ] 히어로 이미지 (16:9)
- [ ] 제목 (Display Serif)
- [ ] 메타: 미술관명(링크) · 기간 · 가격 또는 "Free"
- [ ] 설명 본문 (500~1000자 마크다운)
- [ ] 이미지 갤러리 (최대 8장, 라이트박스)
- [ ] LCP ≤ 2.5s

### US-06 공식 예매 페이지 이동 [M]
**As a** 티켓을 사려는 사용자,
**I want to** "Book Tickets" 버튼 한 번으로 공식 예매 페이지로 이동하고 싶다,
**so that** 제3자 수수료 없이 직접 구매할 수 있다.

- [ ] 버튼은 Primary 스타일, 화면당 1개만
- [ ] 새 탭으로 열림, `rel="noopener noreferrer"`
- [ ] `ticketUrl`이 null일 때 버튼 대신 "Free / No ticket required" 배지
- [ ] 클릭 이벤트는 익명 CTR 집계에만 사용

### US-07 지도에서 위치 확인 [M]
**As a** 전시 결정 후 이동하려는 사용자,
**I want to** "Show on map" 버튼으로 미술관 위치를 바로 볼 수 있다면,
**so that** 교통편을 계획할 수 있다.

- [ ] 상세에서 버튼 탭 시 `/map?gallery=<id>`로 이동
- [ ] 지도는 해당 마커를 중심에 두고 줌 레벨 15
- [ ] 마커가 자동 선택되어 Sheet/Panel에 상세 요약 노출

## 지도

### US-08 미술관 지도 탐색 [M]
**As a** 런던 도보 여행자,
**I want to** 지도에서 주변 미술관을 한눈에 보고 싶다,
**so that** 도보 동선을 짤 수 있다.

- [ ] 런던 중심(51.509, -0.118)을 기본 센터
- [ ] 마커 클러스터링
- [ ] 마커 탭 → Sheet/Panel에 미술관명 + 현재 전시 N개 + "Open details"
- [ ] 현재 위치 권한 수락 시 해당 위치로 센터 이동

### US-09 근처 미술관 [S]
**As a** 위치 권한을 허용한 사용자,
**I want to** 현재 위치 기준 가까운 순으로 정렬할 수 있다면,
**so that** 예상치 못한 방문을 쉽게 할 수 있다.

- [ ] 권한 거부 시 기본 런던 뷰로 fallback, 안내 배너
- [ ] 가까운 순 정렬 시 거리(km) 라벨 표시
- [ ] Geolocation 오류 시 Toast로 사유 안내

## 검색

### US-10 인스턴트 검색 [M]
**As a** 특정 전시/작가를 찾는 사용자,
**I want to** 검색창에서 타이핑 즉시 결과를 보고 싶다,
**so that** 스크롤 없이 빠르게 찾을 수 있다.

- [ ] 입력 후 250ms debounce
- [ ] 결과는 Exhibition / Gallery 섹션으로 구분
- [ ] 0건 시 EmptyState + "기존 목록 둘러보기" 링크
- [ ] 2자 미만일 땐 검색 미발동

### US-11 검색 결과 공유 [C]
**As a** 친구에게 추천하는 사용자,
**I want to** 검색/필터 결과 URL을 공유할 수 있다면,
**so that** 말로 설명하지 않아도 된다.

- [ ] 필터/검색 상태가 URL 쿼리로 저장
- [ ] iOS/Android 기본 공유 시트(Web Share API 지원 시)

## 오류·엣지 상태

### US-12 존재하지 않는 슬러그 [M]
**As a** 오래된 링크를 클릭한 사용자,
**I want to** 404 페이지에서 홈으로 돌아갈 수 있다면,
**so that** 막다른 길에 갇히지 않는다.

- [ ] `/exhibitions/unknown` → 404
- [ ] "Back to home" + "Search for exhibitions" 링크

### US-13 오프라인 상태 [M]
**As a** 지하철 이동 중 연결이 끊긴 사용자,
**I want to** 오프라인 안내 페이지에서도 이전 기록을 볼 수 있다면,
**so that** 앱이 완전히 멈춘 느낌이 들지 않는다.

- [ ] SW가 `/offline`으로 fallback
- [ ] 캐시된 목록/상세는 열람 가능
- [ ] 상단 오프라인 배너

### US-14 이미지 로딩 실패 [S]
**As a** 느린 네트워크 사용자,
**I want to** 이미지가 실패해도 레이아웃이 깨지지 않기를 바란다,
**so that** 스크롤 중 당황하지 않는다.

- [ ] `<Image>`에 width/height 선언 → CLS 방지
- [ ] 실패 시 placeholder (이니셜 또는 로고)

## 접근성·성능

### US-15 키보드 전용 내비게이션 [M]
**As a** 키보드 사용자,
**I want to** 탭/엔터로 모든 핵심 기능을 쓸 수 있기를 바란다,
**so that** 마우스 없이도 앱을 완주할 수 있다.

- [ ] 모든 인터랙션에 visible focus ring
- [ ] Esc로 모달/시트 닫기
- [ ] Skip link ("Skip to main content")

### US-16 스크린리더 호환 [M]
**As a** VoiceOver 사용자,
**I want to** 카드의 제목·기간·링크가 순서대로 읽히기를 바란다.

- [ ] 의미있는 HTML 구조 (h1/h2/h3)
- [ ] 이미지 alt 정책 준수
- [ ] axe 위반 0

### US-17 앱으로 설치 [M]
**As a** 반복 방문자,
**I want to** GalleryUK를 홈 화면에 설치할 수 있기를 바란다,
**so that** 다음엔 브라우저 주소 입력 없이 열 수 있다.

- [ ] Android Chrome 설치 프롬프트 수용
- [ ] iOS Safari "홈 화면에 추가" 안내 도움말
- [ ] 설치된 앱이 standalone 모드로 실행, 스플래시/아이콘 정상

## 사이트 운영

### US-18 전시 자동 만료 [M]
**As a** 사이트 운영자,
**I want to** endDate가 지난 전시를 자동으로 목록에서 숨기고 싶다,
**so that** 오래된 정보로 혼란이 생기지 않는다.

- [ ] 빌드 시 오늘 기준 endDate < today인 항목 필터
- [ ] `/exhibitions/[slug]` 직접 링크는 "종료됨" 배지로 접근 가능

### US-19 정보 제보 받기 [C]
**As a** 사이트 운영자,
**I want to** 사용자가 누락된 전시를 제보할 수 있는 경로를 두고 싶다.

- [ ] 푸터 또는 About 페이지에 제보 이메일/폼 링크
- [ ] v1.0에서는 mailto: 링크로 충분
